# DataVic Victoria (discover-data-vic-gov-au)

DataVic Victoria is a state government open-data portal for Australia running **CKAN**, exposing a standardized catalog API over ~5,585 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/discover-data-vic-gov-au/refs/heads/main/apis.yml)

**Run:** [Capabilities Using Naftiko](https://github.com/naftiko/fleet?utm_source=api-evangelist&utm_medium=readme&utm_campaign=discover-data-vic-gov-au-api-evangelist&utm_content=repo)

## Type
- **kind:** government  ·  **software:** CKAN

## API
- **DataVic Victoria CKAN Action API** — base `https://discover.data.vic.gov.au/api/3/action`. [Docs](https://docs.ckan.org/en/latest/api/) · [Portal](https://discover.data.vic.gov.au)
- Live endpoint verified: `https://discover.data.vic.gov.au/api/3/action/package_search?rows=0` · ~5,585 datasets
